# Hackathon-IBM-ISEN
Hackathon-IBM-ISEN 2021 - Team Inspir'HEI

## Installation:
1) `$ git clone https://github.com/EmileDqy/Hackathon-IBM-ISEN.git`
2) `$ cd Hackathon-IBM-ISEN`
3) `$ npm install`

## To run the app with Expo:

`$ expo start`

## To see the slides:

`run Presentation/index.html in your browser`
